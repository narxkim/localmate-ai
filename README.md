# 🌏 LocalMate AI

> **Spring AI-powered Local Tourism Concierge Platform**

외국인 관광객을 위한 **지역 특화 AI 관광 컨시어지 플랫폼**입니다.

LocalMate AI는 부산·경주를 방문하는 관광객이 관광, 교통, 음식, 문화 정보를 하나의 AI 서비스에서 쉽고 빠르게 얻을 수 있도록 지원합니다.

---

## 📌 Project Overview

여행 중 외국인 관광객은 필요한 정보를 얻기 위해 여러 서비스를 함께 사용해야 합니다.

- 🗺 Google Maps
- 🌐 Papago
- 🏛 Visit Korea
- 📝 Blog
- 🚇 Local Map Services

LocalMate AI는 이러한 불편을 해결하기 위해 **Spring AI 기반의 대화형 관광 컨시어지 서비스**를 제공합니다.

---

## 🎯 Objectives

- AI 기반 관광 컨시어지 서비스 구현
