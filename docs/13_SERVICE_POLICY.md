# 데이터 정책

현재

MVP에서는

직접 구축한 관광 데이터를 사용한다.

향후

- 관광공사 Open API
- 공공데이터 API

를 연동한다.

---

# 개인정보 정책

서비스는

사용자의 개인정보를 저장하지 않는다.

Chat Memory는

대화 품질 향상을 위해

conversationId 기반으로만 관리한다.
