- API Key 외부 관리
- 모바일 화면 지원

# 사용자 요구사항

외국인 관광객은

- 관광지를 추천받고 싶다.
- 이동 방법을 알고 싶다.
- 음식 주문을 쉽게 하고 싶다.
- 한국 문화를 알고 싶다.
